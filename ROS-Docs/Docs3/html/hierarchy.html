 CClientConfigurationsThis structure holds informations relative to the client. It is mostly used for rehab clients. As an example, if you need to modify the max velocity or the retract position, it will be done here. The easiest way to modify the client configuration is to get the current one by calling the function GetClientConfigurations, modify all the parameters you need and send the structure back to the robot by calling the function SetClientConfigurations().
